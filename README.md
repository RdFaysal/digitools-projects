# DigiTools - Premium Digital Products

A modern React-based web application where users can explore and purchase premium digital tools.

---

## Features

- 🛒 Add to cart system
- ❌ Remove from cart
- 🔄 Real-time cart update
- 💰 Total price calculation
- 📱 Fully responsive design
- 🔔 Toast notifications
- 🔘 Tab system (Products & Cart)
- 📌 Sticky Navbar

---

##  Technologies Used

- React.js
- Tailwind CSS
- DaisyUI
- React Icons
- React Toastify
